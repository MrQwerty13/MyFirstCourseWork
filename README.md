# MyFirstCourseWork


Проект представляет собой клиент-серверную систему управления задачами в команде, разработанную на платформе .NET. 
Приложение предназначено для создания проектов, распределения задач между участниками, отслеживания статусов выполнения и контроля сроков. 
Клиентская часть реализована в виде десктопного приложения на C#, взаимодействующего с серверной частью через REST API. 
Серверная часть реализована на ASP.NET и обеспечивает обработку запросов, реализацию бизнес-логики, авторизацию пользователей и взаимодействие с базой данных. 
Система поддерживает разграничение прав доступа и хранение информации в централизованной базе данных.
