# MyFirstCourseWork


The project is a client–server task management system for teams, developed on the .NET platform.

The application is designed to create projects, assign tasks to team members, track execution statuses, and monitor deadlines.

The client side is implemented as a desktop application in C#, which communicates with the server through a REST API.

The server side is built using ASP.NET and is responsible for request processing, business logic implementation, user authorization, and interaction with the database.

The system supports role-based access control and stores information in a centralized database.
